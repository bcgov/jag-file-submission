# Alert

Alert banners notify users of important information or changes on a page.
Typically, they appear at the top of a page.

## When to Use

Telling users important and typically time sensitive information. This includes:

- Success messages
- Error messages
- Warnings
- System statuses
- Important pieces of information

## When not to Use

- In-line error validation on form inputs
- Popular content on your website
- Highlighting content

## API

| Prop    | Type          | Required | Description                                                         | Default |
| ------- | ------------- | -------- | ------------------------------------------------------------------- | ------- |
| type    | string        | true     | indicates whether the alert is a warning, error, or success message |         |
| element | string/object | true     | message to be displayed within the alert                            |         |
